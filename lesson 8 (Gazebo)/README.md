# Урок 8. Gazebo

Раздел зарезервирован для материалов по Gazebo и Gazebo Harmonic в связке с ROS 2 Jazzy.

Планируемые темы:

- установка пакетов Gazebo для ROS 2 Jazzy;
- запуск простого мира;
- подключение URDF/Xacro модели к симуляции;
- сенсоры и плагины;
- мост между Gazebo topics и ROS 2 топиками.

Связанный материал: [TurtleBot3 simulation](<../lesson 10 (turtlebot3 sim)/README.md>).
